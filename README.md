
# Password Generator

A collection of secure Python utilities for password generation 

---

## Password Generator

## What it does
Creates strong, random passwords that are hard to crack.

### Features
- Generates cryptographically secure random passwords
- Customizable length
- Includes uppercase, lowercase, numbers, and special characters
- No patterns or predictable sequences

### Usage
```bash
python password_generator.py
```

### Example Output
```
Your secure password: X7#mK9$pL2@qR5!
```
